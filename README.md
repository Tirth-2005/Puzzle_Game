# Game Description:
## Goal: 
• Rearrange scrambled pieces of an image to restore it to its original form.
## Gameplay:
### • The puzzle board consists of an image divided into tiles (3x3, 4x4, or 6x6 grids).
### • One tile is always blank, and you can slide adjacent tiles into the blank space to rearrange them.
### • Use arrow keys or WASD for movement.
### • Solve the puzzle by arranging all tiles in the correct order.
